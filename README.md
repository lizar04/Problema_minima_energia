La implementación completa de los algoritmos descritos, así como los generadores de problemas prueba y los *scripts* de análisis estadístico, fueron desarrollados en el lenguaje de programación Python. Se utilizaron librerías especializadas como NumPy para la vectorización eficiente del cálculo de energía, PuLP para el modelado de Programación Lineal Entera, y Matplotlib para la visualización de resultados.

El código fuente, los datos de las 30 corridas independientes y las matrices de los grafos generados se encuentran alojados aquí.
